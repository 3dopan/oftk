# Resources Directory

This directory contains application resources.

## Required Icons

- `icon.ico` - System tray icon (normal state)
- `icon_active.ico` - System tray icon (active state)

## Icon Specifications

- Format: ICO (Windows Icon)
- Recommended sizes: 16x16, 32x32, 48x48
- Transparent background
